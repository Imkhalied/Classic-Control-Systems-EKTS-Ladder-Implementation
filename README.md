# Classic Control Systems – EKTS Implementation

## Project Overview
Practical implementation of classical industrial control circuits using relay logic and ladder diagrams. 
The project includes multiple motor control configurations developed and simulated using EKTS software.

## Project Contents
- Direct-On-Line (DOL) motor control
- Forward / Reverse motor control
- Star–Delta starter implementation
- Interlocking and safety circuits
- Timer-based sequential motor control
- Multi-motor coordination logic

## Tools Used
- EKTS (Electrical Control Training Simulator)
- Classical Relay Logic
- Three-phase motor control principles
